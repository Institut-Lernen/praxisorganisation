Excel-Export

Erzeugt die vier Original-Excel-Dateien (gleiche Struktur wie deine Vorlagen) aus den App-Daten. Im finalen Produkt ersetzt die Microsoft-Graph-Sync diesen manuellen Schritt.

Hinweis: Im Prototyp werden die Daten lokal im Browser (localStorage) gespeichert. In der Produktiv-App liegt die Excel in OneDrive/SharePoint und wird via Microsoft Graph API synchronisiert – damit mehrere Therapeut:innen gleichzeitig arbeiten können.
